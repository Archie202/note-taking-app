# Note Taking App

## Description

**Note Taking App** is a responsive web application for managing notes efficiently. With features like rich text editing, categorization with tags, and full-text search, this app provides a comprehensive solution for note-taking. It supports offline use and stores notes locally in the browser, ensuring accessibility across sessions.

## Features

- **Create, Edit, Delete, and Search Notes:** Easily manage your notes with a user-friendly interface.
- **Rich Text Editing:** Add headings, bold/italic text, bullet points, and more.
- **Categorization:** Organize notes with tags.
- **Search Functionality:** Quickly find notes with full-text search.
- **Responsive Design:** Optimized for both desktop and mobile devices.
- **Offline Mode:** Access and manage your notes even without an internet connection.

## Technologies Used

- **HTML** for structuring the content.
- **CSS** for styling and responsive design.
- **JavaScript** for interactive features and note management.
- **Quill.js** for rich text editing.
